# Seonuk Gradient Windows Theme

This folder contains a Windows `.theme` template for pairing Seonuk Gradient with the Windows personalization flow.

## Use

1. Publish the Windows screensaver.
2. Rename `GradientScreenSaver.exe` to `GradientScreenSaver.scr`.
3. Copy it to `C:\Windows\System32\GradientScreenSaver.scr`.
4. Double-click `SeonukGradient.theme`.
5. Run `GradientWallpaper.exe` separately if you also want the static wallpaper behind desktop icons.

Windows theme files can select the screensaver, colors, visual style, and static wallpaper settings. They do not launch the WorkerW wallpaper app, so it remains a separate executable.
